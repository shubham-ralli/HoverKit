About HoverKit

HoverKit is a small, open-source collection of ready-to-use CSS button hover effects. Every effect is pure CSS — no JavaScript, no framework, no build step. Drop in one stylesheet and style any button with a single class name.

It's built to be copy-paste friendly: preview an effect, click it, and paste the snippet straight into your project. The library grows over time as new effects are added, so check back for more.

How to use

  1. Add the stylesheet once — via CDN or the local hoverkit.css file.
  2. Add the btn base class plus any effect class, e.g. btn--lift.
  3. That's it. Recolor with the --btn-bg, --btn-fg and --btn-accent variables.
<!-- Add once in your <head> -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/shubham-ralli/hoverkit@main/hoverkit.min.css" />

<!-- Use any button by class name -->
<button class="btn btn--lift">Click me</button>
